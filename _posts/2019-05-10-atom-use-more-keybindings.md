เอาล่ะ พร้อมแล้วก็มาว่ากันเลยทีเดียว:
- `Ctrl`{: .key}+`Shift`{: .key}+`P`{: .key} คือที่นี่จะมีทุกอย่างของ Keybinding ใน Atom เรียกว่าเป็นที่สุดแล้ว
- `Ctrl`{: .key}+`S` อันนี้น่าจะคุ้นกันดีสำหรับการ Save File นั่นเอง
- `Ctrl`{: .key}+`\` เอาไว้เปิด-ปิด Tree View กัน
- `Alt`{: .key}+`\` ส่วนอันนี้เอาไว้โดดไป-มาเวลาจะให้ Tree View กลับมา Active จริงๆ เค้าเรียกว่า `Toggle Focus` นั่นล่ะถึงจะถูก
- `Ctrl`{: .key}+`W` นี่เอาไว้สำหรับปิดไฟล์ที่กำลังใช้งานอยู่
- `Ctrl`{: .key}+`Tab` อันนี้เอาไว้เลือกวนไปหาไฟล์ที่เปิดใช้งานอยู่ เวลาที่เราเปิดไฟล์ขึ้นมาเยอะๆ
- `Ctrl`{: .key}+`` หรือ `Ctrl`{: .key}+`` 2 อย่างนี้เอาไว้ขยับ cursor ไปทีละคำ หน้า/หลัง ก็แล้วแต่
- `Ctrl`{: .key}+`ฺBackspace` เอาไว้ลบถอยกลับมาทีละคำ
- `Ctrl`{: .key}+`Delete` เอาไว้ลบไปข้างหน้าทีละคำ
-`Ctrl`{: .key}+`Shift`+`K` ลบทั้งบรรทัด
- `Ctrl`{: .key}+`Z` เรื่องลบนี่งงมั๊ย? ลองใช้ดู แล้ว Undo ด้วย `Ctrl+Z` นะพวกเรา
